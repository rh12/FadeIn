Events, Sessions and Scenes
FadeIn uses a simple hierarchy to help you organize your gigs. Let's look at a festival (an Event), which have several days (Sessions), and each day you want to store your work after each soundcheck (Scene).

In this scenario you add a new Event by pressing the '+' button in the Sessions tab and select Event. After customizing various details you create the Event by pressing Done. You can change these details any time by pressing Edit.

On the Event page you can add Sessions one by one. When creating a Session (and any time after that) you can add Equipment to it. Each new Session will have the Equipment of the previous Session preselected for you, and the date is incremented by one day.

On a Session page you can add Scenes for each band. You can assign an Artist to a Scene and select Equipment to use from the Session (or add new ones).
Selecting a Scene loads its default Equipment to the Soundcheck view. If the Equipment has not yet been loaded in this Session, the app loads the Setup Defaults page, where you can customize the default values for each module type.

If you want to view the Scene page instead of loading the Soundcheck view, press the corresponding blue arrow.

You can create a Single Session Event if your Event does not span multiple days, or you are interested in only one act. To do this press the '+' button in the Sessions tab, but select Session instead of Event.
Artists
You can create a new Artist by pressing the '+' button either in the Artists tab or when assigning an Artist to a Scene.

Pressing the Scenes button on an Artist page presents you the list of related Scenes.

Pressing the Info button on a Scene page brings you to the related Artist page.
Venues
You can create a new Venue by pressing the '+' button either in the Venues tab or when setting a Venue for an Event or Session.

Pressing the Events & Sessions button on a Venue page presents you the list of related Events and Sessions.

Pressing the Info button on an Event or Session page brings you to the related Venue page.
Tips
To delete an Event (or any other item), select it from the list, press Edit at the top of its page, then press the Delete button which appears at the bottom.

Your Events and Sessions are ordered by date, but you can reorder your Scenes freely in a given Session by draging them around in Edit mode.

If you use multiple Equipment in a Scene, the first one is loaded by default. On the Scene page you can load any of the Equipment listed, and you can reorder them in Edit mode.

If you want to add several Sessions to an Event first, and add Scenes later, you should add the Sessions in Edit mode. This way the newly added Sessions will not appear automatically. You can also use this method when adding Scenes to a Session.

The values you set on the Setup Defaults page will be used as default values for all Scenes in that Session. If you don't want to use the Setup Defaults feature, you can disable it in the Settings.

When Editing a Scene you deselect an Equipment, all stored values will be deleted for that Equipment (only in the current Scene, of course). Reselecting it will not bring the values back.

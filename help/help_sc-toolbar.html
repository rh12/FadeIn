Back
Closes the Soundcheck view, and brings you back to the previous Clipboard page or the Main Screen.
Channel functions
You can copy all control values of the Active Module and paste them into another one of the same type.
A channel reset clears all control values in the Active Module, and also clears its notes and photos.
Load Values
Loads the values from another Scene of the current Session, or resets the values to the (costumized) defaults.
On the Load Values screen you can decide which Scene to use, and whether to load Channel Names, Control Values or both.
Settings/Help/About
Brings up the Settings. From here, you can also reach the Help and About pages.
